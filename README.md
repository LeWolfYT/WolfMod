# WolfMod

WolfMod is a mod that does a bunch of stuff like discord rpc and other cool stuff.

## Features

Items:<br>
-Debug Item | An item used for testing if the mod is working

Creativetabs:<br>
-WolfTab | A tab containing most of the WolfMod things
